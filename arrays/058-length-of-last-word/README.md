# LeetCode #58: Length of Last Word

## Problem
Return the length of the last word in a string. A word is a sequence of non-space characters.

## Approach
- Skip trailing spaces
- Count characters until the next space or string start

## Time and Space
- Time: O(n)
- Space: O(1)
