\# LeetCode #151: Reverse Words in a String



\## Problem

Reverse the order of words in a string. Words are separated by spaces. Remove leading, trailing, and extra spaces.



\## Approach

\- Trim the string

\- Split into words

\- Reverse and join using a single space



\## Time and Space

\- Time: O(n)

\- Space: O(n)
