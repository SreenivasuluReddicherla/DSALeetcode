# LeetCode #27: Remove Element

## Problem
Given an integer array `nums` and an integer `val`, remove all occurrences of `val` in-place and return the new length.

## Approach
- Use the Two Pointers technique
- Time Complexity: O(n)
- Space Complexity: O(1)
