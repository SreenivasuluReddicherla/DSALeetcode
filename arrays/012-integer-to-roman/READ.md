# LeetCode #12: Integer to Roman

## Problem
Convert an integer (1 to 3999) to a Roman numeral.

## Approach
- Use greedy subtraction
- Store Roman symbols and values in descending order
- While num >= value[i], subtract and append

## Time and Space
- Time: O(1)
- Space: O(1)
