\# LeetCode #219: Contains Duplicate II



\## Problem

Return `true` if any value appears at most `k` distance apart in the array.



\## Approach

\- Use a HashSet and sliding window of size `k`.

\- Check for presence in the window.



\## Time and Space

\- Time: O(n)

\- Space: O(k)
