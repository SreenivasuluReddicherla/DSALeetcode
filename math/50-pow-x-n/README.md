\# LeetCode #50: Pow(x, n)



\## Problem

Implement `pow(x, n)` which calculates x raised to the power n.



\## Approach

\- Use Binary Exponentiation (Divide and Conquer)

\- Handle negative exponents by computing 1 / x^(-n)



\## Time and Space

\- Time: O(log n)

\- Space: O(1)
