\# LeetCode #69: Sqrt(x)



\## Problem

Find the integer part of the square root of a non-negative integer `x`.



\## Approach

\- Use binary search between 1 and x

\- Compare mid \* mid with x

\- Store the last valid result in case of floor rounding



\## Time and Space

\- Time: O(log x)

\- Space: O(1)
