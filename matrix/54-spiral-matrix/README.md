\# LeetCode #54: Spiral Matrix



\## Problem

Given an m x n matrix, return all elements in spiral order.



\## Approach

\- Maintain four boundaries: top, bottom, left, right.

\- Traverse in spiral direction and shrink boundaries accordingly.
