\# LeetCode #392: Is Subsequence



\## Problem

Check if string `s` is a subsequence of string `t`.



\## Approach

\- Use two pointers: one for `s`, one for `t`

\- Traverse `t` and try to match characters of `s` in order



\## Time and Space

\- Time: O(n)

\- Space: O(1)
